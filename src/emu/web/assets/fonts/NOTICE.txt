S60SC.ttf

Series 60 Simplified Chinese system font (family "Sans MT SC 936_S60C").
Contains both embedded bitmap strikes (12/14/16/18/20 ppem, matching the
original S60 on-device look) and scalable outlines for other sizes.

Purpose in EKA2L1 web build: glyph-level fallback font so CJK text in
Symbian apps renders even when the device ROM fonts lack those glyphs
(e.g. western S60v5 firmware such as the Nokia 5320 ROM).

This is a system font extracted from S60 devices; it is bundled here only
to reproduce the original device behaviour for Chinese-language apps.
